# 顶部栏用户菜单箭头设计

## 目标

在后台顶部栏右侧的用户头像区域补充与 `system/pages` 原型一致的下拉箭头，并让箭头状态与现有用户菜单交互保持一致。

## 交互设计

- 箭头位于用户名和角色信息右侧，默认朝下。
- 点击头像、用户信息或箭头所在的整个用户按钮，继续调用现有 `toggleUserMenu`，展开或收起用户菜单。
- 用户菜单展开时，箭头旋转 180 度朝上；菜单关闭时恢复朝下。
- 点击菜单中的“退出登录”继续调用现有 `handleLogout`：关闭菜单、清理登录状态并跳转到登录页。
- 点击用户菜单外部时，菜单与箭头状态同步关闭。

## 实现范围

- 在 `system/frontend/src/layout/AdminLayout.vue` 的用户按钮末尾加入 SVG 折线箭头，并通过 `userMenuOpen` 绑定展开样式。
- 在 `system/frontend/src/assets/styles.css` 中增加箭头颜色、尺寸、旋转过渡和移动端间距样式。
- 更新 `system/frontend/scripts/verify-user-menu.mjs`，验证箭头存在、状态绑定正确，并保留退出登录行为检查。

## 验收标准

- 顶部栏头像框右侧显示与原型一致的小箭头。
- 用户菜单关闭时箭头朝下，展开时箭头平滑旋转朝上。
- 点击整块用户按钮可开关菜单；点击“退出登录”可正常退出并跳转登录页。
- 前端构建与用户菜单静态验证通过。
