- [需求基线](01-项目总览/需求基线.md)：确认当前正式功能范围、技术选型、模块边界和排除项。

## 文档维护原则

- 项目功能范围以项目书和需求基线为准。
- 数据库结构以 `../sql/market.sql` 为唯一正式来源。
- 后端接口权限以代码中的 `@RequireRoles` 和认证拦截器为准。
- 前端权限只负责菜单、路由和按钮展示，不能替代后端权限校验。
