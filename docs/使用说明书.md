# 使用说明书

## 1. 登录说明

系统默认初始化两个演示账号：

| 用户名 | 密码 | 角色 |
|---|---|---|
| `admin` | `admin123` | 管理员 |
| `user` | `user123` | 普通用户 |

系统不会在任何页面展示密码或密码哈希。

## 2. 角色说明

管理员：

- 可管理用户和角色分配。
- 可新增、编辑、删除商品。
- 可维护库存上下限。
- 可执行入库、出库和盘点。
- 可查看报表和系统信息。

普通用户：

- 不可进入用户管理。
- 可查看商品、库存、报表和系统信息。
- 可新增入库、出库。
- 不可新增盘点。
- 不可维护商品资料和库存上下限。

## 3. 操作说明

商品管理：

- 管理员可新增商品。
- 商品新增后系统自动创建初始库存记录。
- 商品页面不提供库存数量维护入口。

库存管理：

- 可查看当前库存和预警状态。
- 管理员可维护库存上下限。
- 禁止直接手工修改库存数量。

入库管理：

- 新增入库记录后，后端调用库存模块增加库存。
- 操作人为空时使用当前登录用户。

出库管理：

- 新增出库记录前，后端校验库存是否充足。
- 库存不足时出库失败。

盘点管理：

- 管理员可提交实际库存。
- 系统记录系统库存、实际库存和差异。
- 后端调用库存模块将当前库存调整为实际库存。

报表统计：

- 只读展示库存、入库、出库和库存变化趋势。
- 不提供新增、编辑、删除功能。

系统信息：

- 展示系统名称、版本、后端技术、数据库和运行环境。
- 不实现复杂配置中心或日志管理。
